A fork of Ami-Sampler with a new sample start-point parameter, wider codec support (I wanted to load .qoa, .opus, .flac, .ape, .ogg, .aiff, and .mp3 files), the custom cursor removed (it was wonky for me in Wine), and the internal patch save/load stripped out since I only use my DAW's preset function.
